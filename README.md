# jd-browser-debs
Build Debian Package for JDownloader Browser

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- [libjd-utils-java](https://github.com/yadickson/jd-utils-debs)

**Tested**

- Debian jessie
